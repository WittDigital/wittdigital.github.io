WittBot 隱私權政策

最後更新日期：2026年3月2日

1. 我們收集的資料類型

本機器人僅會處理實現功能所必需的最小化資料：

  • 使用者識別資訊： 使用者 ID (User ID)、使用者名稱 (Username)。
  • 伺服器資訊： 伺服器 ID (Guild ID)、頻道 ID (Channel ID)、身分組 ID。
  • 指令內容： 當您使用 / 指令或特定前綴指令時，本機器人會處理您輸入的參數。

2. 資料的使用方式

收集到的資料僅用於以下用途：

  • 功能運作： 確保指令能正確回應至對應的頻道或使用者。
  • 權限驗證： 確認使用者是否有權限執行特定管理指令。
  • 開發調試： 在發生錯誤時記錄必要的 Error Log 以修復問題。

*我們承諾：除非法律要求，否則絕不會將您的資料出售、出租或分享給任何第三方。

3. 資料儲存與安全

大部分指令處理皆在記憶體中完成，訊息處理後即釋放。若功能涉及伺服器設定（如自定義歡迎訊息），相關資料將儲存於受保護的資料庫中。我們不會主動收集您的密碼、信用卡資訊或任何現實生活中的敏感個資。

4. 使用者權利與資料刪除

您可以隨時從伺服器中移除本機器人以終止資料處理。若您希望手動刪除本機器人儲存的特定資料，請透過下方聯絡方式聯繫開發者。

5. 聯絡方式

如果您對本隱私政策有任何疑問，或需要處理資料刪除請求：

  • 年齡限制： 您必須符合 Discord 規定的最低年齡限制。
  • 合法使用： 您不得利用本機器人進行任何違法、騷擾、威脅、散布惡意軟體或違反 Discord 規範的行為。
